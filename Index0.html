Основні теги,які використовуються при створенні сайту: Тег HTML вказує початок і кінець HTML документа. Тег HEAD (заголовок HTML-документа) містить інформацію, що відноситься до документа в цілому. Тег BODY охоплює ту частину HTML-документа (текст, зображення та елементи формування), яка буде видно користувачеві. </ title> Тег TITLE встановлює заголовок HTML-документа, що виводиться в рядку заголовка вікна броузера. <body bgcolor=?> Атрибут BGCOLOR встановлює колір фону HTML-документа. Колір може бути вказаний за допомогою назви або коду. <body text=?> Атрибут TEXT встановлює колір для всього тексту HTML-документа. Колір може бути вказаний за допомогою назви або коду. <body link=?> Атрибут LINK встановлює колір всіх посилань HTML-документа, які ще не були активізовані. <body vlink=?> Атрибут VLINK встановлює колір всіх посилань HTML-документа, які вже були одного разу активізовані. Теги елементів тексту <h1> </h1> Тег H1 визначає заголовок першого рівня (найбільші літери у заголовку). Всього є шість рівнів заголовків. <h6> </ h6> Тег H6 визначає заголовок шостого рівня (найдрібніші букви в заголовку). <i> </ i> Тег I встановлює відображення тексту курсивом. <strong> </ strong> Тег STRONG встановлює відображення тексту в жирному накресленні. <font size=?> </ font> Атрибут SIZE тега FONT встановлює відносний розмір шрифту. Список можливих значень складається з позитивних і негативних чисел від 0 до 7. <font color=?> </ font> Атрибут COLOR тега FONT встановлює колір тексту. <p> </ p> Тег P встановлює початок і кінець абзацу. <p align=?> Атрибут ALIGN тега P встановлює режим вирівнювання тексту: ліворуч (LEFT), вправо (RIGHT) або по центру (CENTER). <br> Тег BR встановлює перенесення рядка (навіть якщо текст триває за тегом). <ol> </ ol> Тег OL («ordered list» — впорядкований список) встановлює режим послідовної нумерації елементів, розміщених усередині списку. <ul> </ ul> Тег UL («unordered list» — невпорядкований список) встановлює режим відображення тексту у вигляді маркованого списку. <li> </ li> Тег LI встановлює режим відображення тексту у вигляді пункту списку. Такий список може бути неврегульованим, упорядкованим або маркованим. <div align=?> Тег DIV використовується для структурування та форматування тексту. Атрибут ALIGN тега DIV встановлює режим вирівнювання тексту. Можливо вирівнювання вліво (LEFT), вправо (RIGHT) або по центру (CENTER). Теги графічних елементів <img src=»name»> Тег IMG встановлює режим відображення графічного файлу. <img src=»name» align=?> Атрибут ALIGN тега IMG встановлює режим вирівнювання зазначеного зображення у відведеному йому просторі. Можливо вирівнювання вліво (LEFT), вправо (RIGHT), по центру (CENTER), вниз (BOTTOM), вгору (TOP), посередині (MIDDLE). <img src=»name» border=?> Атрибут BORDER тега IMG встановлює режим відображення рамки навколо зазначеного зображення. Теги для створення таблиць <table> </ table> Тег TABLE встановлює початок і кінець таблиці. Всі теги, що визначають структуру таблиці, повинні розташовуватися між тегами TABLE. <tr> </ tr> Тег TR визначає рядок таблиці. <td> </ td> Тег TD визначає колонку таблиці. Текст, укладений між тегами TD («table data» — табличні дані), відображається всередині одного осередку. <th> </ th> Тег TH встановлює режим відображення тексту у вигляді заголовка таблиці. <table border=?> Атрибут BORDER тега TABLE встановлює ширину рамки таблиці в пікселах. <table width=?> Атрибут WIDTH тега TABLE встановлює ширину таблиці в пікселах або у відсотках від ширини області перегляду. Теги для створення фреймів <frameset> </ frameset> Тег FRAMESET встановлює область екрану, яка повинна використовуватися броузером для відображення кадрів. <frame name=»name»> Атрибут NAME встановлює ім’я вікна FRAME, що складається з букв і цифр, на яке може посилатися гіперпосилання в іншому HTML-документі. <frame scrolling=VALUE> Атрибут SCROLLING тега FRAME встановлює, чи відображати всередині кадру смуги прокручування. (Можливі значення атрибута: YES, NO або AUTO). <frame noresize> Атрибут NORESIZE тега FRAME встановлює режим, при якому розміри описуваного кадру не можуть бути змінені користувачем.
